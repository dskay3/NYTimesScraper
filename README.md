## The Application
This application searches for articles from the NY Times homepage website.


## Technologies
+ ReactJS
+ Express
+ Morgan
+ Body-Parser
+ NodeJS
+ Javascript
+ HTML
+ CSS
+ Materialize
+ Bootstrap
+ Webpack
+ Mongoose
+ Axios
+ React-Router
+ REST API
  + NY Times API for displaying NY Times articles
