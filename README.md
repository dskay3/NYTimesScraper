## The Application
This application scrapes articles from the NY Times homepage website.

## Technologies
+ ReactJS
+ Express
+ Morgan
+ Body-Parser
+ NodeJS
+ Javascript
+ HTML
+ CSS
+ Materialize
+ Bootstrap
+ Webpack
+ Mongoose
+ Axios
+ React-Router
